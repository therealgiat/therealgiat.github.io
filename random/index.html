Sentrale komponenter i datamaskiner!

Hva er en datamaskin?

  • Kompleks maskin som består av mange forskjellige komponenter, som har en spesifikk rolle og jobber sammen for at datamaskinen kjøres effektivt

CPU (Central Processing Unit):

  • Hjernen i datamaskinen som utfører de fleste beregningene
  • Leser instrukser og utfører det i en bestemt rekkefølge
  • Hastigheten til CPU måles i gigahertz (GHz), antall instrukser per sekund

RAM (Random Access Memory):

  • Midlertidig lagringsplass for data som prosessoren bruker aktivt
  • Rask tilgangshastighet
  • Mer RAM gir mulighet for flere applikasjoner samtidig
  • Tømmes når datamaskinen slås av

Harddisk:

  • Primære lagringsenhet
  • Lagres all data og informasjon
  • Mekaniske (HDD), spinner eller solid state (SSD), chip ingen bevegelige deler
  • SSD er raskere og mer pålitelige, men også dyrere

Operativsystem (OS):

  • Programvaren som styrer alle de andre programmene
  • Utfører grunnleggende oppgaver som å administrere filer, kjøre programmer, kontrollere utstyr som skrivere og mus

Eksempler på OS:

  • Windows: brukervennlig, bred kompatibilitet med mange forskjellige programvarer, populært blant gaming pga. bredt utvalg av spill
  • Linux: stabilitet, sikkerhet, tilpasningsdyktig, bredt utvalg av systemer (superdatamaskin til smarttelefon), populært blant programmerere og IT-profesjonelle pga. fleksibilitet og kontroll
  • MacOS: elegant design, stabilitet og sømløs integrasjon med andre Apple-produkter og tjenester (MacBook, iPhone og iPad kommuniserer med hverandre), populært blant grafisk design, videoredigering og musikkproduksjon

Motherboard (hovedkort):

  • Kobler alle komponentene sammen
  • Tillater kommunikasjon imellom

Grafikkort:

  • Spesialisert prosessor designet for å håndtere grafikk og bildebehandling
  • Viktig for applikasjoner som krever høy ytelse, som videoredigering og spill

Strømforsyning:

  • Gir alle komponentene strøm
  • Konverterer strøm fra veggen til den type strøm komponentene trenger
